# USAGE 
Fork or download this repo , move inside and run:

```
npm install
npm start
```
Open development build on http://localhost:3000/

**TO DEPLOY** :
```
npm run build
npm install -g serve
serve -s build
```
Open production build on http://localhost:5000/

Or copy to a Web Server
